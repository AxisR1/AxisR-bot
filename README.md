# AxisR-bot プライバシーポリシー
### 本ポリシーは、AxisR（以下「当Bot」といいます）が取得する情報の範囲、利用目的、および管理方法について定めるものです。

## 第1条（取得情報の範囲と利用目的）
当Botは、サービスの提供・改善のため、以下の情報を取得および利用します。
### 1.取得情報の範囲
・ユーザーID、メッセージ内容および履歴
・サーバーID、チャンネルID、サーバー設定等の関連情報
・その他、当Botの機能提供に不可欠なデータ
### 2.情報の取得方法
・当Botが稼働するプラットフォーム（Discord等）を通じて、機能実行時に自動的に取得します。
### 3.利用目的
・当Botの機能提供およびユーザーサポートのため
・サービスの改善、新機能の開発、および利用状況の分析のため
・不正利用の防止およびセキュリティ維持のため
### 4.情報の保持期間
・メッセージ内容等の動的なデータは、通常短期間（1〜2日程度）で破棄または上書きされます。ただし、設定情報やセッションデータなど、機能維持に必要な情報は、必要に応じて内部データベースや、外部データベースへ保存されます。

## 第2条（情報の管理と保護）
### 1.安全管理措置
・当Botは、取得した情報の漏洩、紛失、改ざん、または不正アクセスを防止するため、適切なセキュリティ対策を講じます。
### 2.第三者提供の禁止
・法令に基づく場合を除き、ユーザーの同意なく取得した情報を第三者に提供することはありません。
### 3.ユーザーの権利
・ユーザーは、自己の個人情報について、開示・訂正・削除・利用停止を求めることができます。希望される場合は、第4条の連絡先までお問い合わせください。

## 第3条（ポリシーの変更）
1.当Botは、必要に応じて本ポリシーを変更することがあります。
2.変更後のポリシーは、当Bot内または指定の場所（公式サーバー等）に掲示された時点から効力を生じるものとします。
